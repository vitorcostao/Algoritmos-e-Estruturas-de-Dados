# Codificação

## Definição

**Codificação** é o processo de transformar informações de uma forma para outra, geralmente para facilitar **armazenamento**, **transmissão** ou **processamento**.  
Pode ser aplicada a textos, números, imagens, sons e outros tipos de dados.

---

## Para que serve

A codificação é utilizada para diversos fins, como:

- **Armazenamento eficiente:** Representar dados de forma compacta para economizar espaço.  
- **Transmissão segura e rápida:** Facilitar o envio de dados em redes ou sistemas de comunicação.  
- **Processamento de dados:** Permitir que computadores entendam e manipulem diferentes tipos de informações.  
- **Proteção de informações:** Em alguns casos, a codificação também serve para **criptografia** ou **ocultação de dados**.

---

## Tipos comuns de codificação

- **Codificação de texto:** ASCII, UTF-8, Unicode  
- **Codificação numérica:** Binário, hexadecimal, decimal  
- **Codificação de áudio e vídeo:** MP3, AAC, MPEG  
- **Codificação de imagens:** JPEG, PNG, BMP  
- **Codificação de compressão de dados:** Huffman, LZW, Run-Length Encoding (RLE)

---

## Exemplo

Representando números em diferentes codificações:

| Número decimal | Binário | Hexadecimal |
|----------------|---------|-------------|
| 10             | 1010    | A           |
| 255            | 11111111| FF          |

Cada forma de codificação mantém a mesma informação, mas representa os dados de maneira diferente.

---

## Conclusão

A codificação é essencial para **comunicação e armazenamento de dados**. Ela garante que informações possam ser **interpretadas, transmitidas e armazenadas** de forma eficiente e segura.
